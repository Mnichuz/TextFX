### Example usage:

```python
print(GenText(['bold','green', 'underline'], 'TextFX!'))
```

### To print all text options:
```python
TextOptions()
```

### All commands:
* __TextOptions()__
* __GenText(attributes,text)__
* __Space(x)__
* __GenLine(length)__
